Example

Token: s3617_to_ara3
Bucket: clviz-bucket
Num points: 3000

This should take about 20 minutes to run.
Detailed instructions can be found below.

Token is the token of the brain to retrieve from ndstore.

Bucket is the aws S3 bucket to save the outputs to.

Num Points is the number of points to be represented in the plots. The lesser the number of points, the faster each graph loads.

Access key id is your aws access key id.

Secret access key is your aws secret access key.

Detailed Instructions

  • Input your token, s3 bucket, and number of points in the corresponding text boxes.
  • Press enter and the web service will take approximately 20-30 minutes to process. The processing is all done through aws batch, and will upload the results to the specified s3 bucket when done, and the results will be viewable then.
  • In the details page for each submission, you will find links that you can click on to download the corresponding file.
  • If you wish to process another token, just go to the compute page again and fill in the form again.
