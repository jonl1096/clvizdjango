Example

Token: Aut1367
Orientation: LPS
Num points: 10000

This should take about 5 to 10 minutes to run.
Detailed instructions can be found below.

Token is the token of the brain to retrieve from ndstore.

Orientation is an all caps 3 letter orientation of the brain. Information about brain orientation can be found here: http://www.grahamwideman.com/gw/brain/orientation/orientterms.html

Num Points is the number of points to be represented in the plots. The lesser the number of points, the faster each graph loads.

Detailed Instructions

  • Input your token and the orientation in the corresponding text boxes. The orientation for the AUT brains is "RPS".
  • Press enter and the web service will take approximately half an hour to process. Please make sure not to close the website while data is being processed.
  • When finished an html page will pop up with two different kinds of clickable interfaces. The buttons will link you to another html page with a plot or graph while the clickable links will download the plot or graph.
  • If you wish to process another image, simply refresh the page and follow the steps above once more!
